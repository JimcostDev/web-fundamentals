Etiquetas para texto

Recordemos la semántica: son reglas que debemos seguir para nuestro posicionamiento en los navegadores.

Énfasis

strong: Se utiliza para enfatizar texto importante.

em: Se utiliza para enfatizar texto con un significado especial.

Otro tipo de formato

Small: Texto más pequeño.

Marked: Texto marcado.

Deleted: Texto tachado.

Inserted: Texto subrayado.

Loremsubscript: Subíndice.

Loremsuperscript: Superíndice.
